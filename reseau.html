C'est quoi ?

Ce projet consiste en la création d'un script php, utilisant des commandes bash, auquel on fournit des images, et le rôle de ce script est de déterminer si le format, les dimensions ou bien le poids de ces fichiers sont compatibles pour une utilisation dans un site web. Ce script permet donc d'automatiser toutes ces tâches, en n'ayant qu'à entrer qu'une commande dans un terminal pour que le traitement des fichiers se fasse.
